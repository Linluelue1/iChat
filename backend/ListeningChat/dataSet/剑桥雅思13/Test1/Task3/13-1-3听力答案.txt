剑桥雅思13 Test1 Section3听力答案解析
第21题答案：A
对应原文：there’s an optional module on seed structure and function in the third year that I might do, so I thought it might be useful for that
答案解析：A选项module原词出现，related topic对应seed structure and function由此锁定答案。B选项由Emma提出，但被Jack所否定。C选项Jack自己否定，因此统统排除。
第22题答案：C
对应原文：
EMMA:
OK. We’d need to allow time for the seeds to come up.
JACK:
That should be fine if we start now. A lot of the other possible experiments need quite a bit longer
答案解析：这道题为反义替换。原文中强调其他实验需要更长的时间，即反过来说明该实验用时较短，由此确定C为正确答案。
第23题答案：B
对应原文： But we need to be sure we’re the only ones doing it
答案解析：三个选项在录音中虽然都有提到，但A选项和C选项是已经确定的事情，只有B是需要跟导师进一步核实的，因此锁定答案。
第24题答案：C
对应原文：I found it quite hard to follow – lots about the theory, which I hadn’t expected
答案解析：Emma提出，这本书很难懂，有太多关于理论的东西，这是她实现没有想到的。然后男性也表达了认同，符合题干中agree的要求，由此确定答案为C。
第25题答案：B
对应原文：His analysis of figures comparing the times of the fires and the proportion of seeds that germinated was done in a lot of detail – very impressive
答案解析：A选项Diagram是另一本书里的，C选项错在surprising。B选项analysis是原词，thorough对应in a lot detail，由此锁定答案。
第26题答案：G
对应原文：And obviously they mustn’t all be the same size. So how many sorts do we need? About four different ones
答案解析：听到size意识到答案即将到来，sort与type同义替换，由此确定G为正确答案。
第27题答案：C
对应原文：Then, for each seed we need to find out how much it weighs, and also measure its dimensions
答案解析：这道题需要根据then来确定答案所在句子，不过流程图的考点主要就是顺序连接词，相信大家事先一定都有准备。weights与weight同义替换，由此确定C为正确答案。
第28题答案：H
对应原文：That’ll be quite time-consuming. And we also need to decide how deep we’re going to plant the seeds
答案解析：听到also和decide意识到答案即将到来，how deep与depth同义替换替换，由此确定H为正确答案。
第29题答案：A
对应原文：
JACK:
OK. So then we get planting. Do you think we can plant several seeds together in the same plant pot?
EMMA:
No, I think we need a different one for each seed
答案解析：Jack提议，要不要把所有的种子都种到一个盆里，女性提出每个种子都要不同的盆，由此确定A为正确答案。
第30题答案：E
对应原文：Then we see if our plants have come up. and write down how tall they’ve grown
答案解析：录音中提到，在种子长出来之后，需要记录他们的高度，tall与height同义替换，由此确定E为正确答案。
